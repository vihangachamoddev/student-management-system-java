# Student Management System (Java)

## 📌 Project Overview

This is a simple **Java Console Application** developed using Object-Oriented Programming (OOP) concepts.
The system helps to manage student records efficiently.

---

## 🚀 Features

* Add new student records
* Search student by ID
* Update student details
* Delete student records
* Display all students

---

## 🛠️ Technologies Used

* Java
* OOP Concepts
* Array / ArrayList
* Console-based UI

---

## 📂 Project Structure

```
StudentManagementSystem/
│
├── Main.java
├── Student.java
└── StudentService.java
```

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/student-management-system.git
```

2. Open project in any Java IDE (IntelliJ / Eclipse / VS Code)

3. Run `Main.java`
